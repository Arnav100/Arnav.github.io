Background

This is a blog made for recording and sharing work I've done on various programming projects and possibly events I find important. After creating a blog for my school's FRC team, Team 4930 The Electric Mayhem, I felt motivated to make my own blog so that I could experiment more with web development. I do not expect this site to get a lot, if any, network traffic; its real purpose is to record my thought process on the projects I create and allow me to self-educate myself on web design. If you have stumbled across this website and would like to ask questions, provide advice, or would like me to work on a project for you, please do not hesitate to reach out.
